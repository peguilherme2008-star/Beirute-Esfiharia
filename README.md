# Beirute-Esfiharia
A proposta foi criar uma plataforma de delivery para a Beirute Esfiharia, unindo tecnologia, usabilidade e identidade visual para oferecer uma experiência prática e intuitiva ao usuário.
O sistema foi pensado para contar com:
 🍽️ Visualização clara e organizada do cardápio
 🛒 Carrinho de compras dinâmico e fácil de usar
 💳 Múltiplas formas de pagamento (PIX, cartão e dinheiro)
 📦 Acompanhamento do pedido em tempo real
 📱 Interface moderna e totalmente responsiva
 ⚙️ Painel administrativo para gestão de pedidos, produtos e usuários
O projeto ainda está em fase de desenvolvimento, mas já apresenta uma estrutura sólida, com fluxos bem definidos, banco de dados modelado e uma interface alinhada à proposta da marca.
Em breve, pretendemos implementar novas funcionalidades e melhorias para tornar a experiência ainda mais completa.
